# 高榮企業工會 · 網站與工具

純前端靜態網站；所有小工具都在使用者自己的裝置上運算，不上傳伺服器。

## 頁面
- `gaorong.html` — 工會網站（對外：介紹／入會／常見問答／加入 Line）
- `union.html` — 幹部功能表（內部工具入口）
- `id-mark.html` — 證件影本加註記工具（拍照／裁切／去反光／A4 列印）
- `finance.html` — 工會透明帳本（`finance-law.js`、`laws.json`）
- `receipt.html` — 單據快拍
- `privacy.html` — 個人資料保護政策

## 部署（GitHub Pages）
Settings → Pages → 由 `main` 分支發布即可。首頁可指向 `gaorong.html`。
